# HTML

html não é layout, é semântica.
é para descrever o significado do conteúdo.


## TAGS

<nav> navegação
<section> seção temática
<article> conteúdo independente
<aside> informação complementar
<div> apenas um agrupamento

<div>
    <button>A</button>
    <button>B</button>
</div>


## ESTRUTURA

<html>
    <head>
        metadados: <meta>
        título: <title>
        dependências (css): <link>
    </head>
    <body>
        componentes <p></p>
        aplicação: <script>
    </body>
</html>
