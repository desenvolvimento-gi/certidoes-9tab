# Visão de Carreira

## Propósito

Projetar e implementar soluções tecnológicas que melhorem a gestão de pequenas e médias empresas.

A tecnologia não é um fim em si mesma.

Ela é um meio para tornar processos mais simples, dados mais úteis e organizações mais eficientes.

---

# Identidade Profissional

Minha formação e experiência principal são em Engenharia de Produção.

Minha atuação combina:

- consultoria de gestão;
- modelagem de processos;
- análise de dados;
- automação;
- desenvolvimento de soluções.

Não tenho como objetivo migrar para uma carreira de desenvolvedor full-stack.

Meu objetivo é desenvolver autonomia suficiente para construir aplicações que complementem minha atuação como consultor.

# Visão de Longo Prazo

Construir uma consultoria tecnológica especializada em sistemas de gestão para pequenas e médias empresas.

Projetos podem envolver:

- modelagem de processos;
- digitalização;
- automação;
- engenharia de dados;
- dashboards;
- aplicações administrativas;
- integrações entre sistemas.

O diferencial não será uma tecnologia específica.

O diferencial será a capacidade de compreender o negócio e construir a solução mais adequada.
